**Sugestão de estudo:** leia um arquivo, abra o `.py` correspondente no editor, rode o comando de teste, só depois passe ao próximo.

**Quando pedir ajuda à IA:** traga contexto — “estou na Fase 4, alterei a função X, o erro é Y”. Assim você aprende em vez de colar código sem entender.
